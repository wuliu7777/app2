# 🚀 项目启动联调指南

本指南将帮助你快速在本地将前端（Flutter）与后端（Python FastAPI）同时运行起来，实现全功能联调。

---

## 🟢 启动后端服务 (Python 真实解析引擎)

后端负责处理实际的视频解析工作（利用 `yt-dlp` 与 `you-get` 双引擎）。

**1. 打开一个全新的终端（Terminal 或 PowerShell）**
2. 进入后端目录：**

cd d:\桌面\kaifa\app\backend


**3. 运行服务：**

uvicorn main:app --reload

> **✅ 成功标志**：终端显示 `Uvicorn running on http://127.0.0.1:8000`。
> *注：保持这个终端窗口不要关闭。当你修改后端代码时，它会自动重启。*

---

## 🔵 启动前端服务 (Flutter 绝美 UI 端)

前端负责展示极具质感的 UI 并与用户交互。

**1. 打开另一个全新的终端（Terminal 或 PowerShell）**
**2. 进入前端目录：**

cd d:\桌面\kaifa\app\flutter_app


**3. 运行常驻的热更新 Web 服务：**

flutter run -d web-server --web-port 8080

> **✅ 成功标志与说明**：
> 1. 终端会提示你在 `http://localhost:8080` 启动了服务。你可以**自己手动用浏览器打开这个网址**。
> 2. **常驻优势**：现在就算你**关闭了浏览器网页，后端的命令行也不会停止**！你可以随时重新打开网页继续访问。
> 3. **如何热更新**：当你在开发环境修改了 Dart 代码后，只需要在这个终端里按一下键盘上的 `r` 键，代码就会热更新。然后你在浏览器里按 F5 刷新一下页面，就能立刻看到最新的效果，简直不要太爽！

1. 确保两个终端都在运行（一个跑着 uvicorn，一个跑着 flutter）。
2. 在自动弹出的浏览器网页中，随便输入一个你复制的**抖音/B站真实视频链接**。
3. 点击“批量解析”，你将看到绝美的等待动画，并在解析完成后看到真实的视频标题及播放选项。
4. 如果输入 `ok`，则前端不会请求真实后端，而是直接展示快速测试用例（用于检查 UI 动画效果）。
